 # Password Strength Checker 🔐

## Description

The **Password Strength Checker** is a web-based application that evaluates the security level of a password in real time.
It analyzes the password based on several validation rules such as length, uppercase letters, numbers, and special characters. The application provides visual feedback through a strength indicator bar and rule validation list.

---

## Features

* Real-time password strength analysis
* Strength indicator bar (Weak / Medium / Strong)
* Password visibility toggle (Show / Hide password)
* Validation rules for secure passwords
* Interactive and responsive user interface

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or run it using Live Server in VS Code.
4. Enter a password to see the strength analysis.

---

## Password Rules Checked

* Minimum 8 characters
* At least one uppercase letter
* At least one number
* At least one special character
